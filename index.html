Projects

Order Up! A drag and drop matching game that teaches basic food words in Spanish and French.

I collaborated with a partner to create an educational game in Typescript for my Educational Game Development class March-May 2020.
The game was meant for 5th to 6th graders and introduces them to basic food words in Spanish and French.
Playing as a fast food worker, you have to read customer's orders, then drag and drop food from a conveyor belt into a bag,
matching the correct items. The game begins with the orders in English, then each level introduces words from the target language,
until the order is only in the target language. The goal of this game is to introduce children to simple vocabulary in other languages
to help build a foundation for learning languages in the future.




Estuary Escapades: A multi part game about wildlife in Delaware estuaries.

I worked with four other people to develop an educational game in Java utilizing MVC design patterns for
my Software Engineering class October-December 2019. We worked with a client to create a game about Delaware estuaries for her company
that educates children about estuaries and wildlife conservation. The game consisted of three minigames, each about a minute long,
with a cumulative scoring system. In the first game players have to match animals with their description; in the second, you play as
an animal from the first game where you dodge predators in the estuary; and in the third, you play as a person on the shore picking up
trash using proper tools and receptacles. The goal of the game was to teach children in a quick and memorable way about the animals living
in Delaware estuaries and what can be done to protect them.
